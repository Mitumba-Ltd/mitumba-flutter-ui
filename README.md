# Mitumba Flutter UI

> Mitumba design system for Flutter — reusable widgets for Kenya's second-hand fashion marketplace.

[![pub package](https://img.shields.io/pub/v/mitumba_ui.svg)](https://pub.dev/packages/mitumba_ui)

## Install

```yaml
dependencies:
  mitumba_ui: ^0.1.0
```

## Usage

```dart
import 'package:mitumba_ui/mitumba_ui.dart';

// Use any component
ListingCard(
  title: 'Vintage Leather Jacket',
  price: 3500,
  imageUrl: 'https://...',
  onTap: () {},
);
```

## Components

See the [Widgetbook](https://mitumba-flutter-ui.vercel.app) for live previews.
